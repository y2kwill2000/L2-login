THREAD [STRING]

Calls the given function named [STRING1] on a new thread. All variables are guaranteed thread safe.

Threads sleep independent of each other. Each thread has a unique stack and does NOT share memory/variables with any other thread. End a thread by using the RETURN VOID statement.

Global variables can be used to communicate between threads.

EX:
PRINT_TEXT "11-1"
THREAD t2
PRINT_TEXT "11-2"
THREAD t3
PRINT_TEXT "11-3"
CALL t4
PRINT_TEXT "11-4"
PRINT_TEXT "11-5"
END_SCRIPT

FUNCTION t2
PRINT_TEXT "22-1"
PRINT_TEXT "22-2"
PRINT_TEXT "22-3"
PRINT_TEXT "22-4"
PRINT_TEXT "22-5"
RETURN VOID

FUNCTION t3

PRINT_TEXT "33-4"
PRINT_TEXT "33-5"
RETURN VOID

PRINT_TEXT "44-1"
PRINT_TEXT "44-2"
THREAD t2
