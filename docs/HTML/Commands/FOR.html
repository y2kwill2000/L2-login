FOR [V] [INT] [INT] [INT]

If the variable [V1] does not exist it is created. Otherwise, it is just changed to an int.

[INT2] = starting value
[INT3] = ending value
[INT4] = step (a negative value will make it step backwards)

EX: FOR A 0 10 1
EX: PRINT_TEXT "A = <&A&>"
EX: NEXT

---

NEXT

Causes the associated FOR loop to increase the variable by step. If this new value is greater (or lower if negative) or equal to the end value, it will exit the for loop.

See above for an example.
