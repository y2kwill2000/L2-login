DEFINE [STRING] [STRING] [STRING]

Declares a global variable of type [STRING1] named [STRING2] with default value [STRING3].

Being global... this variable can be accessed by all stack levels.

Built in variable types:
  • NULL
  • INT
  • DOUBLE
  • STRING
  • FILEREADER**
  • FILEWRITER**
  • ARRAYLIST*
  • SORTEDLIST8
  • StACK
  • QUEUE
  • BYTEBUFFER

    *For both ARRAYLIST and SORTEDLIST [STRING3] must resolve to an INT and is the default size of the collection.

    **FILEREADER and FILEWRITER can only open files contained in the Scripts\Files\ folder

    EX: DEFINE_GLOBAL INT MOB_COUNT 0 EX: DEFINE_GLOBAL STRING MOB_S1 "zero" EX: DEFINE_GLOBAL STRING MOB_S2 "<&MOB_COUNT&>"
