SAY_TEXT [INT] [STRING] ([STRING])

[INT1] the channel to say the text in.
[STIRNG2] the text to say.
([STRING3]) the person to send the text to. Only needed when sending private messages.

There are script global variables to use for the channels:
  • CHANNEL_ALL
  • CHANNEL_SHOUT
  • CHANNEL_PRIVATE
  • CHANNEL_PARTY
  • CHANNEL_CLAN
  • CHANNEL_GM
  • CHANNEL_PETITION
  • CHANNEL_PETITIONREPLY
  • CHANNEL_TRADE
  • CHANNEL_ALLY
  • CHANNEL_ANNOUNCEMENT
  • CHANNEL_BOAT
  • CHANNEL_PARTYROOM
  • CHANNEL_PARTYCOMMANDER
  • CHANNEL_HERO

    Ex: SAY_TEXT "<&CHANNEL_ALL&>" "this is a test of the dereferenced global channl value... easy isn't it? ^^"
    Ex: SAY_TEXT "<&CHANNEL_PRIVATE&>" "you are a poopoo head" "notspider"
