SET was deprecated in v358! Use = now!

SET [V] [STRING]

Used to set the value of variable V1 after creation when not assigning another variable to it.

The type of the variable is used to determine the type of the value STRING2. Be careful not to assign strings to ints or doubles, or any other weird combo. You may not get the results you expect.

SET only works with INT, DOUBLE and STRING types.

EX: DEFINE INT A 10
EX: SET A 100
EX: DEFINE DOUBLE B 10.1
EX: SET B 11.11
EX: DEFINE STRING STR "hello poopy heads"
EX: SET STR "i'm no poopy head
