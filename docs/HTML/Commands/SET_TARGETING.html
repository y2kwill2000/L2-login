SET_TARGETING [STRING] [V]

Used to set the parameters for the target nearest functions. Gives the paramater of type [STRING1] the value of [V2]

All parameters default to their 0 (ZERO) state.

Parameters:

TYPE
0 = npcs
1 = pcs
2 = both

ATTACKABLE
0 = attackable
1 = invincible(for npcs), invisible(for pcs)
2 = both

ALIVE
0 = alive
1 = looks dead (might be really dead... or just using fakedeath)
2 = both

INBOX
0 = inside border polygon
1 = outside the box
2 = both
Remember a bounding polygon is defined as at least 3 points... If you have less than 3 points in your border, everything will be considered "inside" the box.

COMBAT
0 = not in combat
1 = in combat
2 = both

ZRANGE
variable set at the distance

More items will come in later releases.

EX: SET_TARGETING TYPE ONE
EX: SET_TARGETING COMBAT TWO
EX: SET_TARGETING ZRANGE my_defined_dist
