MATH [V] [V] [STRING] ([V] [V] [V})

V1 = Destination var
V2 = Source var 1
STRING3 = any of a set of values listed below
V4 = Source var 2
V5 = Source var 3
V6 = Source var 4

  • ADD V1 = V2 + V4
  • SUBTRACT V1 = V2 - V4
  • MULTIPLY V1 = V2 * V4
  • DIVIDE V1 = V2 / V4
  • SQRT V1 = sqrt( V2 )
  • POW V1 = V2 ^ V4
  • EQUAL V1 = V2
  • MOD V1 = REM( V2 / V4 )
  • ABS V1 = ABS( V2 )
  • SIN V1 = SIN( V2 )
  • SINH V1 = SINH( V2 )
  • ASIN V1 = ASIN( V2 )
  • COS V1 = COS( V2 )
  • COSH V1 = COSH( V2 )
  • ACOS V1 = ACOS( V2 )
  • TAN V1 = TAN( V2 )
  • TANH V1 = TANH( V2 )
  • ATAN V1 = ATAN( V2 )
  • LOG V1 = LOG (base V4) of V2
  • REPLACE V1 = V2 where all instances of V4 have been replace with V5
  • TO_UPPER V1 = V2 capitalized
  • TO_UPPER_INVARIANT V1 = V2 capitalized based on the rules of the invariant culture
  • TO_LOWER V1 = V2 lower cased
  • TO_LOWER_INVARIANT V1 = V2 lower cased based on the rules of the invariant culture
  • SUBSTRING V1 = V2 starting at index V4 and reading V5 characters
  • LENGTH V1 = length of the string V2
  • PADLEFT V1 = V2 padded with the char V5, V4 times on the left
  • PADRIGHT V1 = V2 padded with the char V5, V4 times on the right
  • TRIM V1 = V2 with all white space chars removed from the front
  • TRIMSTART V1 = V2 with all instaces of V4 removed from the front
  • TRIMEND V1 = V2 with all instances of V4 removed from the end
  • INDEXOF V1 = index of V4 in V2 starting at index V5 and through the next V6 characters
  • LASTINDEXOF V1 = last index of V4 in V2 starting at index V5 and through the next V6 characters
  • GET_HEX V1 = string hex value of V2

    Types are maintained EXCEPT for sin/cos/tan operations. INTs are typecast to DOUBLE and the result is typecast back to an INT

    EX: DEFINE INT A 0
    EX: DEFINE INT B 20
    EX: DEFINE INT C 30
    EX: MATH A B ADD C
