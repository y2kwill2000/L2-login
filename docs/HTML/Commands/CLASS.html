CLASS [STRING] [V]

[STRING1] is the name of the class.

[V2] is the type of the parent class. This is for inheritence. Use NULL if you want to inherit from nothing.

Start a class with the CLASS command and end it with the END_CLASS command. Only 1 class may be in a file.

You may include other classes from a class. When you declare a class you must specify the parent class. Use NULL if you have no parent class.

Class functions can be declared using PUBLIC, PRIVATE and PROTECTED instead of the FUNCTION command.

All variables must be defined inside a VAR_START - VAR_END block. You may only have 1 VAR_START - VAR_END block per class.

Should make more sense if you look at the following code blocks:

my_class.l2c
---
CLASS Super null
VAR_START

PUBLIC INT A 100
PUBLIC DOUBLE B 10.1

VAR_END

END_CLASS
---

child_class.l2c
---
INCLUDE Tests\my_class.l2c

CLASS NewSuper Super
VAR_START

PUBLIC INT C 50

VAR_END

END_CLASS
---

Once you have declared a class, you can create types of it using the DEFINE command just as if it was a built in type.

Take a look at class_test.l2s in the Scripts\Tests\ folder for a better example.
