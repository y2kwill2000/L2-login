FUNCTION [STRING] [INT] ([V] [V] ... )

STRING1 is the word the function will be called by. It is not case sensitive (ie. MyFuNc = MYFUNC = myfunc)

INT2 is an integer for how many variables the function takes.

Following is a list of variables as long as the count.

EX: FUNCTION MYFUNC 0
EX: FUNCTION MYFUNC2 1 VAR1
EX: FUNCTION MYFUNC3 2 VAR1 VAR2
