REGISTER_SHORTCUT [INT] [INT] [INT]

[INT1] is the type of shortcut. [INT2] is the slot and the [INT3] the id of the thing to be stuck in there.

Use the steps described in USE_SHORTCUT to calculate the slot.

Please note, the IDs are the unique type id for items.

Lets say we want to add an item of with a unique id of 1000 to slot 5 on the second bar.

EX: REGISTER_SHORTCUT "<&SHORTCUT_ITEM&>" 16 1000
