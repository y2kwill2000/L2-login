LOCK [STRING]

This will place a lock on the string specified. As long as you hold the lock on this string... no other thread will be able to lock on it and will be forced to sleep. The string is NOT case sensitive.

---

UNLOCK [STRING]

This will release the lock held on the string. This means that another thread could grab ahold of it now. The string is NOT case sensitive.

SPECIAL NOTE: ALWAYS ALWAYS ALWAYS release your locks in the reverse order you created them!

Such as:
LOCK A
LOCK B
LOCK C
UNLOCK C
UNLOCK B
UNLOCK A
