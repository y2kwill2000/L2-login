DEFINE [STRING] [STRING] [STRING]

Declares a variable at the current stack height of type [STRING1] named [STRING2] with default value [STRING3].

Built in variable types:
  • NULL
  • INT
  • DOUBLE
  • STRING
  • FILEREADER **
  • FILEWRITER **
  • ARRAYLIST *
  • SORTEDLIST *
  • StACK
  • QUEUE
  • BYTEBUFFER *

    *For ARRAYLIST, SORTEDLIST and BYTEBUFFER [STRING3] must resolve to an INT and is the default size of the collection.

    **FILEREADER and FILEWRITER can only open files contained in the Scripts\Files\ folder

    EX: DEFINE INT MOB_COUNT 0
    EX: DEFINE STRING MOB_S1 "zero"
    EX: DEFINE STRING MOB_S2 "<&MOB_COUNT&>"
    EX: DEFINE FILEREADER filein "my_data.txt"
    EX: DEFINE BYTEBUFFER bbuff 128
