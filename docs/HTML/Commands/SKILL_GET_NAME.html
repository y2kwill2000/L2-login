SKILL_GET_NAME [VSTRING] [INT] [INT]

Stores the name of the SKILL with type id INT2 at level INT3 in variable VSTRING1 as a string.

The variable will be changed to a string regardless of what it's type was before the call.

EX: DEFINE STRING name "none"
EX: SKILL_GET_NAME name 100
