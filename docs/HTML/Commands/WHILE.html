WHILE [V] [conditional] [V]

while the conditional is true the code between the WHILe and associated WEND will be run. When the WEND is reached it will jump back up to the WHILE. If the conditional is false... the code will jump to the line following the associated WEND.

Like IF, this function needs actual variable names and not values for strong type comparisons.

EX: DEFINE INT A 10
EX: DEFINE INT B 20
EX: WHILE A < B
EX: //put code here
EX: WEND

---

WEND

Ends a WHILE conditional statement. If the conditional is still true... the code will jump back up to the WHILE. If the conditional is false... code will continue on the next line.
