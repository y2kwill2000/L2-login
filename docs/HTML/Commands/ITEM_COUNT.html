ITEM_COUNT [V] [VINT]

Stores the count of items of type [VINT2] (type ID, not unique ID) into V1 cast as the same type as V1.

If the item is not in your inventory... then 0 is stored to V1.

EX: DEFINE STRING Itm "100"
EX: DEFINE STRING ItmI 100
EX: DEFINE STRING Dest "ItmCnt"
EX: DEFINE INT ItmCnt 0
EX: ITEM_COUNT ItmCnt "<&Itm&>"
EX: ITEM_COUNT ItmCnt <&ItmI&>
EX: ITEM_COUNT ItmCnt 100
EX: ITEM_COUNT "<&Dest&>" "<&Itm&>"
EX: ITEM_COUNT "<&Dest&>" "<&ItmI&>"
EX: ITEM_COUNT "<&Dest&>" 100
