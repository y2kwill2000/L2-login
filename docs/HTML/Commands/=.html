=

= [v]
= [OP] [V]
= [V] [OP] [V]

Used for assignment to variables.

Supported unary operations:

VARIABLE_DEFINED or []
SQRT
ABS
SIN
SINH
ASIN
COS
COSH
ACOS
TAN
TANH
ATAN
-

Supported binary operations:
+
-
*
/
^
%
LOG
||
&&
|
&
<<
>>
XOR
==

At this time parenthesis () are not supported with the = operator.

EX: A = VARIABLE_DEFINED B
EX: A = SQRT B
EX: A = B + C
EX: A = A * B
