SWITCH [V]

Will perform a series of IF/ELSE/ENDIF sort of blocks based on the passed var [V1]. SWITCH works with INT, DOUBLE and STRING types.

See the Test\switch_test.l2s for a good example.

Use ENDSWITCH to terminate your SWITCH statement

---

CASE [V]

A potential value for a SWITCH condition.

CASE can fall through to other cases. C/C++ style.

Use a break statement to pop out of a SWITCH

---

DEFAULT

The "other" case. This conditional will be true if no other case is.
