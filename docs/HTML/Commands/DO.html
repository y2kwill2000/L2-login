DO

starts a DO LOOP conditional block

---

LOOP [V] [conditional] [V]

While the conditional is true, jump back up to the DO and run the code again. Think of a DO/LOOP as a backwards WHILE/WEND. DO/LOOP runs the code... then checks the conditional. WHILE/WEND checks the conditional and then runs the code.

Actual variables need to be passed instead of just the value.

EX: DEFINE INT A 10
EX: DEFINE INT B 20
EX: DO
EX: LOOP A < B
EX: DO
EX: LOOP A != B
