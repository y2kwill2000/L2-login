NPC_DIALOG [STRING]

Selects the dialog at the given link. Use the Debug Mode and manually select a link or button once to find the text that needs to be passed to this command.

This command supports both text links and buttons (think clan hall buffer).

EX: NPC_DIALOG "test.htm"
EX: DEFINE STRING talk "test.htm"
EX: NPC_DIALOG "<&talk&>"
