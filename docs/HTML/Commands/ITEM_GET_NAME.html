ITEM_GET_NAME [VSTRING] [INT]

Stores the name of the ITEM with type id INT2 in variable VSTRING1 as a string.

The variable will be changed to a string regardless of what it's type was before the call.

EX: DEFINE STRING name "none"
EX: ITEM_GET_NAME name 100

---

ITEM_GET_DESC [VSTRING] [INT]

Stores the description of the ITEM with type id INT2 in variable VSTRING1 as a string.

The variable will be changed to a string regardless of what it's type was before the call.
