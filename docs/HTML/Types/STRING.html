STRING

A unicode string.

Member Functions:
.TYPE
Returns the global STRING

.CLASSNAME
Returns a string "STRING"

.CLONE
Returns a copy of the variable.

.GET_HEX
Returns the string version of the hex value of this variable. 2 bytes per character delimited with 2 bytes of 0x00

.LENGTH
Returns an int containing the length of the string

.COUNT
Same as .LENGTH

.REVERSE
Reverses the character order in the string.

.TO_UPPER
Returns the string where all characters have been converted to uppercase using the local system's locale uppercase rules.

.TO_UPPER_INVARIANT
Returns the string where all characters have been converted to uppercase using the invariant culture rules (English).

.TO_LOWER
Returns the string where all characters have been converted to lowercase using the local system's locale uppercase rules.

.TO_LOWER_INVARIANT
Returns the string where all characters have been converted to lowercase using the invariant culture rules (English).

.TRIM
Returns the string where all whitespace characters have been removed from the the start and end of the string.

.LASTINDEXOF [v]
Returns the int location of the last index of the string version of the value of V1.

.INDEXOF [v]
Returns the int location of the first index of the string version of the value of V1.

.TRIMEND [v]
Returns the string where string version of the value of V1 has been trimmed from the end.

.TRIMSTART [v]
Returns the string where string version of the value of V1 has been trimmed from the start.

.CONTAINS [v]
Returns the global TRUE or FALSE depending if the string version of the value of V1 was contained in the string.

.SUBSTRING [v] [v]
Returns the string starting from the integer version of the value of V1 to the integer version of the value of V2.

.REPLACE [v] [V]
Returns the string where the string version of the value of V1 was contained in the string was replaced with the string version of the value of V2.

.STARTSWITH [v]
Returns the global TRUE or FALSE depending if the string version of the value of V1 was at the start of the string.

.ENDSWITH [v]
Returns the global TRUE or FALSE depending if the string version of the value of V1 was at the end of the string.
