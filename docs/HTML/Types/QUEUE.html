QUEUE

A queue of variables. First in first out (FIFO)

Member Functions:
.TYPE
Returns the global QUEUE

.CLASSNAME
Returns a string "QUEUE"

.CLONE
Returns a copy of the variable.

.CLEAR
Removes all elements from the queue.

.COUNT
Returns an int for how many elements are in the queue.

.LENGTH
Same as .COUNT

.PUSH [V]
Pushes [V1] onto the top of the queue.

.ADD [V]
Same as .PUSH [V]

.PEEK
Returns the variable at the head of the queue.

.POP
Returns the variable at the head of the queue and removes it from the queue.
