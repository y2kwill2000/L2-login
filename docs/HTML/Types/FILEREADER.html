FILEREADER

An object used to read from text files.

Member Functions:
.TYPE
Returns the global FILEREADER

.CLASSNAME
Returns a string "FILEREADER"

.CLONE
Returns a copy of the variable.

.CLOSE
Close the opened file. No further data may be read.

.READ [v]
Reads a line from the file which is then converted to the type of [v1] and stored in [v1]. Only INT, DOUBLE and STRING types may be read.
