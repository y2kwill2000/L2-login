FILEWRITER

An object used to write to text files.

Member Functions:
.TYPE
Returns the global FILEWRITER

.CLASSNAME
Returns a string "FILEWRITER"

.CLONE
Returns a copy of the variable.

.FLUSH
Forces the file to commit any cached writes to disk.

.CLOSE
Close the opened file. No further data may be written.

.WRITE [v]
Writes the value of variable [v1] to the file. Only INT, DOUBLE and STRING types may be written.
