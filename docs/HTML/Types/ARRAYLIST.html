ARRAYLIST

An arraylist of variables. Also called a vector, array, or list.

Member Functions:
.TYPE
Returns the global ARRAYLIST

.CLASSNAME
Returns a string "ARRAYLIST"

.CLONE
Returns a copy of the variable.

.CLEAR
Removes all elements from the arraylist.

.COUNT
Returns an int for how many elements are in the arraylist.

.LENGTH
Same as .COUNT

.REVERSE
Reverses the order of the elements in the arraylist.

.[VINT]
Returns the element at index [VINT1].

.ADD [V]
Puts [V1] at the end of the arraylist.

.PUSH [V]
Same as .ADD [V]

.REMOVE [INT]
Removes the element at location [INT1]. The data is shifted left accordingly.

.REMOVEAT [INT]
Same as .REMOVE
