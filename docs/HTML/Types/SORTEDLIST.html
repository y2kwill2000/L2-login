SORTEDLIST

A sortedlist of variables.

Member Functions:
.TYPE
Returns the global SORTEDLIST

.CLASSNAME
Returns a string "SORTEDLIST"

.CLONE
Returns a copy of the variable.

.CLEAR
Removes all elements from the sortedlist.

.COUNT
Returns an int for how many elements are in the sortedlist.

.LENGTH
Same as .COUNT

.[VINT]
Returns the element at index [VINT1].

.[STRING]
Returns the element with key [STRING1].

.ADD [V] [STRING]
Puts [V1] with key [STRING2] into the sortedlist.

.PUSH [V] [STRING]
Same as .ADD [V1] [STRING2]

.GET_KEY [VINT]
Returns the string key at index [VINT1]

.REMOVE [STRING]
Removes the element with key [STRING1]. The data is shifted left accordingly.

.REMOVEAT [INT]
Removes the element with index [INT1]. The data is shifted left accordingly.
.CONTAINS_KEY [STRING]
Returns the global TRUE or FALSE depending if the key [STRING1] exists.
