STACK

A stack of variables. First in last out (FILO)

Member Functions:
.TYPE
Returns the global STACK

.CLASSNAME
Returns a string "STACK"

.CLONE
Returns a copy of the variable.

.CLEAR
Removes all elements from the stack.

.COUNT
Returns an int for how many elements are in the stack.

.LENGTH
Same as .COUNT

.PUSH [V]
Pushes [V1] onto the top of the stack.

.ADD [V]
Same as .PUSH [V]

.PEEK
Returns the variable on the top of the stack.

.POP
Returns the variable on the top of the stack and removes it from the stack.
