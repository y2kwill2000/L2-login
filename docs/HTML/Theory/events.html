Events are similar to event handlers in c#, vb, and several other languages. You setup a function to be called when an event fires.

Unlike some other languages...there can only be one function to be called per event. You could always make that function start up other threads if you want.

Every event will provide you will different variables that you can read to find out various information. Please see the script event thread to learn more.

SCRIPTEVENT_CHATTOBOT

INT MESSAGETYPE
STRING TARGETNAME
STRING MESSAGE

Chat sent from the real client through L2.Net or typed into the chat box in L2.Net and prefaced with a minus (-) will trigger this event. Text prefaced with minus minus (--) will be converted to a single minus, passed to the server, and will not trigger this event.
